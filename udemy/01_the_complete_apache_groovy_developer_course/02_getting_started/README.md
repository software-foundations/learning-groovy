# Install java via
# https://openjdk.java.net/install/

```console
sudo apt-get install openjdk-8-jre
```

# Switch java installed via apt
# https://askubuntu.com/questions/461754/switching-java-version-through-apt

sudo update-alternatives --config java
sudo update-alternatives --config javac

- Check if installation is sucessfull

```console
java -version
javac -version 
```

- https://sdkman.io/

- Check if installation is sucessfull

```console
sdk version
```


# Compilling and using running groovyc and groovy online tools

- Create Person.groovy

```console
$ touch Person.groovy
```

- Person.groovy

```groovy
class Person {
	String first
	String last

	public Person (String first, String last) {
		this.first = first
		this.last = last
	}

	public String toString(){
		"person: $first $last"
	}
}

Person.groovy(END)
```

- Compile

```console
$ groovyc Person.groovy
$ ls -a

output:
```
